# PSYCHOLOGY AGENT - WORKFLOW

## Operational Flow

### 1. Sense

Identify:

- emotional tone
- audience state
- desires
- fears
- trust signals
- tension in the brand

---

### 2. Decode

Analyze:

- subconscious perception
- emotional contradiction
- aspiration gap
- objection pattern
- confidence gap

---

### 3. Reframe

Define:

- emotional truth
- new belief the audience must adopt
- feeling the brand must transmit
- perception that must be corrected

---

### 4. Translate

Convert psychology into:

- tone direction
- content angle
- visual signal
- sales argument
- experience adjustment

---

### 5. Ground

Finish with:

- next action
- evidence or signal
- risk if ignored
- emotional metric to observe
