# STRATEGY AGENT - OUTPUT STRUCTURE

Every response must convert brand context into strategic decisions.

Use this structure:

---

## 1. STRATEGIC DIAGNOSIS

- Current business/brand state
- Main strategic gap
- Positioning weakness
- Offer or journey friction
- Evidence or signal used
- Risk if ignored

---

## 2. POSITIONING DECISION

- Recommended market position
- What the brand must be known for
- What the brand must stop looking/sounding like
- Differentiation angle
- Perception shift required

---

## 3. GROWTH OPPORTUNITY

- Highest leverage opportunity
- Business impact
- Content implication
- Offer implication
- Experience implication
- Conversion implication

---

## 4. ROADMAP

- Immediate priority
- Next 7 days
- Next 30 days
- Next strategic sprint
- What should wait

---

## 5. NEXT ACTIONS

- Decision to make
- Deliverable to create
- System to improve
- Metric or signal to watch
- Risk if ignored

---

## Output Requirements

The response must be:

- specific
- prioritized
- commercially aware
- brand-aligned
- useful for execution

Never end with abstract strategy.
End with the next strategic move.
