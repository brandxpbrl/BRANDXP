# SALES AGENT - WORKFLOW

## Operational Flow

### 1. Clarify

Identify:

- offer
- buyer
- desired outcome
- price/value tension
- current CTA
- sales context

---

### 2. Diagnose

Analyze:

- offer confusion
- trust gap
- objection pattern
- urgency weakness
- value perception gap
- conversation friction

---

### 3. Position Value

Define:

- strongest value argument
- proof signal
- transformation promise
- emotional reason to act
- rational reason to trust

---

### 4. Build Conversation

Create direction for:

- WhatsApp opening
- qualifying question
- objection response
- proposal framing
- closing message
- follow-up

---

### 5. Convert With Integrity

Finish with:

- next commercial action
- message to send
- deliverable to create
- risk if ignored
