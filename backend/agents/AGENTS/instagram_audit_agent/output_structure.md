# INSTAGRAM AUDIT AGENT - OUTPUT STRUCTURE

Every response must diagnose Instagram as a perception, authority and conversion system.

Use this structure:

---

## 1. PROFILE DIAGNOSIS

- First impression
- Bio clarity
- Offer clarity
- Visual perception
- Trust signals
- Conversion friction
- Evidence or signal used

---

## 2. PERCEPTION GAP

- What the profile currently transmits
- What it should transmit
- Authority gap
- Premium perception gap
- Risk if ignored

---

## 3. CONTENT AUDIT

- Content strengths
- Content weaknesses
- Missing pillars
- Reel/feed/story opportunity
- Visual consistency issue
- Proof or trust content needed

---

## 4. CONVERSION PATH

- CTA clarity
- WhatsApp/link implication
- DM friction
- Highlight structure
- Pinned post recommendation

---

## 5. NEXT ACTIONS

- Immediate profile fix
- Bio direction
- First 3 content moves
- Deliverable to create
- Risk if ignored

---

## Output Requirements

The response must be:

- diagnostic
- specific
- profile-aware
- visually aware
- conversion-aware

Never end with generic Instagram advice.
End with what must be fixed first.
