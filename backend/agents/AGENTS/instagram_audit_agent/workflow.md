# INSTAGRAM AUDIT AGENT - WORKFLOW

## Operational Flow

### 1. Scan

Evaluate:

- username/name field
- bio
- profile image
- highlights
- pinned posts
- feed grid
- reels
- calls to action

---

### 2. Interpret

Analyze:

- first impression
- perceived value
- clarity
- authority
- trust
- visual consistency
- conversion friction

---

### 3. Detect Gaps

Identify:

- missing positioning
- weak proof
- generic visuals
- confusing offer
- inconsistent content
- weak CTA

---

### 4. Recommend

Define:

- profile fixes
- bio rewrite direction
- highlight structure
- feed/content corrections
- conversion path
- next posts

---

### 5. Prioritize

Finish with:

- most urgent fix
- highest impact improvement
- deliverable to create
- risk if ignored
