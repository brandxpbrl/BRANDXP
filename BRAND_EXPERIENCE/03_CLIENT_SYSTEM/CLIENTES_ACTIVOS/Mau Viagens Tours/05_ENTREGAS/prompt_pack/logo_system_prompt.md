# LOGO GENERATION PROMPT — CLIENTE

## Uso

Este prompt se usa después de cargar o leer los archivos principales del cliente:

1. `identity_cliente.md`
2. `LATEST_ANALYSIS.md`
3. `full_brand_experience_prompt_pack.md`
4. Paleta de color aprobada, si existe
5. Ruta creativa elegida, si existe

El objetivo es generar un sistema de logo coherente con la identidad estratégica del cliente, no un logo decorativo ni genérico.

---

# PROMPT MAESTRO PARA GENERAR LOGO

Actúa como director creativo senior, diseñador de identidad visual premium, estratega de marca y especialista en sistemas visuales.

Usa toda la información disponible en la carpeta del cliente como contexto maestro, especialmente:

- identidad de marca
- diagnóstico actual
- promesa de marca
- público ideal
- personalidad verbal
- dirección visual
- storytelling
- paleta cromática
- tensión de percepción
- diferenciadores
- estilo cultural del sector
- ruta creativa seleccionada

Antes de diseñar, interpreta la entidad de marca.

No generes un logo genérico.  
No uses íconos stock.  
No uses símbolos obvios sin transformación conceptual.  
No hagas solo una estética linda: el logo debe traducir estrategia, percepción y recordación.

---

## OBJETIVO

Crear un sistema de logo profesional para la marca:

**[NOMBRE DEL CLIENTE]**

El logo debe aumentar:

- claridad
- autoridad
- deseo
- confianza
- recordación
- diferenciación
- valor percibido

Debe funcionar como una identidad real, aplicable en Instagram, WhatsApp, sitio web, presentaciones comerciales, favicon, perfil, piezas impresas y contenido digital.

---

## INTERPRETACIÓN ESTRATÉGICA OBLIGATORIA

Antes de generar el logo, define internamente:

1. Qué debe dejar de parecer la marca.
2. Qué debe empezar a parecer.
3. Qué emoción debe transmitir.
4. Qué percepción debe instalar en el público.
5. Qué símbolos son coherentes con su universo.
6. Qué símbolos deben evitarse por ser genéricos.
7. Qué nivel de lujo, cercanía o autoridad necesita.
8. Qué tipo de sistema visual puede sostener la marca en el tiempo.

---

## DIRECCIÓN DE LOGO

Diseña un sistema de logo con:

1. Logo principal.
2. Versión horizontal.
3. Versión compacta.
4. Símbolo o isotipo.
5. Versión monocromática.
6. Versión para fondo claro.
7. Versión para fondo oscuro.
8. Versión usable como foto de perfil o favicon.

El símbolo debe ser simple, memorable y reconocible.

Debe poder vivir solo sin depender siempre del nombre.

---

## CRITERIOS VISUALES

El logo debe tener:

- composición limpia
- jerarquía clara
- buen uso del espacio negativo
- lectura inmediata
- forma memorable
- equilibrio entre estrategia y belleza
- estética profesional
- aplicación real en marca
- buena reducción a tamaños pequeños

Evitar:

- exceso de detalles
- ilustración compleja
- clipart
- íconos genéricos
- estética de plantilla
- efectos innecesarios
- mockups falsos
- letras difíciles de leer
- símbolos que no tengan relación con la promesa de marca

---

## TIPOGRAFÍA

Crear una dirección tipográfica coherente con la personalidad de la marca.

La tipografía debe expresar:

- confianza
- claridad
- autoridad
- personalidad propia
- buena lectura comercial

Puede ser:

- serif premium
- sans serif moderna
- wordmark personalizado
- mezcla editorial y contemporánea

Pero debe justificarse por la identidad del cliente, no por tendencia.

---

## COLOR

Usar la paleta aprobada del cliente si existe.

Si el cliente tiene una paleta definida, respetarla como sistema principal.

Si no existe paleta definida, proponer una paleta estratégica con:

- color base
- color secundario
- color de acento
- color neutro
- versión monocromática

Los colores deben responder a la percepción deseada de la marca.

---

## FORMATO DEL RESULTADO VISUAL

Generar una presentación tipo board de identidad visual, con fondo limpio y composición premium.

El board debe incluir:

1. Logo principal grande.
2. Símbolo / ícono.
3. Versión horizontal.
4. Versión monocromática.
5. Versión invertida sobre fondo oscuro.
6. Paleta cromática en pequeños círculos o bloques.
7. Nombre de la marca correctamente escrito.

No agregar textos largos dentro de la imagen.

Puede incluir etiquetas pequeñas como:

- Principal
- Ícono
- Horizontal
- Monocromático
- Revertido
- Paleta

Pero mantener una estética limpia y profesional.

---

## ESTILO DE RENDER

Estilo:

- vectorial
- limpio
- premium
- editorial
- moderno
- profesional
- minimalista pero con carácter
- sin 3D
- sin sombras pesadas
- sin mockup realista
- sin fotografía
- sin exceso de textura

El resultado debe parecer una lámina de presentación de identidad hecha por un estudio senior de branding.

---

## PROMPT FINAL PARA IMAGEN

Crear un board profesional de identidad visual para la marca **[NOMBRE DEL CLIENTE]**.

Usar toda la información estratégica del cliente como contexto: identidad, promesa, público, tono, dirección visual, paleta, diferenciadores, storytelling y tensión de percepción.

Diseñar un sistema de logo premium, claro, memorable y aplicable.

El logo debe comunicar la esencia de la marca, aumentar confianza, deseo, autoridad y recordación.

Crear un símbolo simple y reconocible, con buen uso del espacio negativo, que pueda funcionar solo como ícono, favicon o foto de perfil.

Diseñar también el wordmark con el nombre exacto:

**[NOMBRE DEL CLIENTE]**

Incluir en la composición:

- logo principal
- versión horizontal
- versión compacta
- símbolo / isotipo
- versión monocromática
- versión sobre fondo oscuro
- paleta cromática
- aplicación limpia tipo brand board

---

## NEGATIVE PROMPT

No usar logos genéricos.  
No usar íconos stock.  
No usar aviones, globos, pasaportes, flechas o símbolos obvios salvo que sean transformados de forma conceptual y única.  
No usar mockups de pared, tarjetas o papelería.  
No usar fotografía.  
No usar 3D.  
No usar sombras pesadas.  
No usar efectos brillantes innecesarios.  
No usar tipografías ilegibles.  
No escribir mal el nombre de la marca.  
No agregar slogans inventados.  
No agregar textos explicativos largos dentro de la imagen.  
No romper la paleta aprobada del cliente.  
No crear una imagen decorativa sin sistema de marca.
