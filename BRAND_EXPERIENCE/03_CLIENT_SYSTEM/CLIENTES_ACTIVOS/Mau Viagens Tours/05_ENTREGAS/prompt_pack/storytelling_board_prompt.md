# STORYTELLING VISUAL BOARD PROMPT — CLIENTE

## Uso

Este prompt se usa para generar una imagen tipo **Storytelling Visual Board** a partir de toda la información estratégica del cliente.

Debe usarse después de cargar o leer:

1. `identity_cliente.md`
2. `LATEST_ANALYSIS.md`
3. `full_brand_experience_prompt_pack.md`
4. `VISUAL_UNIVERSE_BOARD_PROMPT.md`
5. `LOGO_GENERATION_PROMPT.md`, si existe
6. Paleta aprobada
7. Ruta creativa seleccionada
8. Análisis visual del cliente o imagen de referencia, si existe

El objetivo es generar una lámina visual completa, profesional y estratégica que muestre la historia, percepción, emoción, universo visual, tono, pilares y dirección de contenido de la marca.

No debe ser solo un moodboard.  
Debe parecer una página de brand book premium, storytelling estratégico y dirección de arte.

---

# PROMPT MAESTRO PARA STORYTELLING VISUAL BOARD

Actúa como director creativo senior, estratega de marca, diseñador de identidad visual premium, director de arte, especialista en storytelling visual y arquitecto de sistemas de marca.

Usa toda la información disponible en la carpeta del cliente como contexto maestro.

Analiza especialmente:

- identidad de marca
- diagnóstico actual
- promesa central
- propósito
- público ideal
- tensión de percepción
- energía emocional
- tono de voz
- personalidad verbal
- valores
- diferenciadores
- storytelling
- universo visual
- paleta cromática
- símbolos permitidos
- símbolos prohibidos
- aplicaciones prioritarias
- ruta creativa seleccionada

Antes de diseñar la imagen, interpreta la entidad de marca.

El resultado debe ser una imagen tipo **brand storytelling board**, similar a una lámina estratégica de identidad visual premium, con múltiples secciones organizadas, textos cortos, íconos, imágenes, paleta, composición editorial y narrativa visual.

---

## OBJETIVO

Crear un **Storytelling Visual Board profesional** para la marca:

**[NOMBRE DEL CLIENTE]**

La imagen debe comunicar visualmente:

- quién es la marca
- qué promete
- qué emoción transmite
- qué experiencia vende
- qué percepción debe instalar
- cómo debe verse
- cómo debe sentirse
- qué historia debe contar
- qué códigos visuales debe repetir
- cómo debe aparecer en Instagram, reels, web, WhatsApp y piezas comerciales

El board debe transformar la estrategia en una lámina visual clara, premium y aplicable.
