# VISUAL UNIVERSE BOARD PROMPT — CLIENTE

## Uso

Este prompt se usa para generar el universo visual completo de una marca después de cargar o leer los archivos principales del cliente:

1. `identity_cliente.md`
2. `LATEST_ANALYSIS.md`
3. `full_brand_experience_prompt_pack.md`
4. `LOGO_GENERATION_PROMPT.md`, si existe
5. Paleta aprobada, si existe
6. Ruta creativa elegida, si existe

El objetivo es crear un **Visual Universe Board** que traduzca la identidad estratégica en un sistema visual aplicable a redes, web, piezas comerciales, contenido, campañas y experiencia de marca.

No se trata de hacer un moodboard lindo.  
Se trata de construir un universo visual reconocible, coherente y repetible.

---

Actúa como director creativo senior, estratega de marca, diseñador de identidad visual premium, director de arte y especialista en storytelling visual.

Usa toda la información disponible en la carpeta del cliente como contexto maestro, especialmente:

- identidad de marca

- público ideal
- personalidad verbal

- diferenciadores
- storytelling

- tensión de percepción

- símbolos prohibidos
- ruta creativa seleccionada

## INTERPRETACIÓN ESTRATÉGICA OBLIGATORIA

Antes de generar el board, definir internamente:

1. Qué debe dejar de parecer la marca.
2. Qué debe empezar a parecer.
3. Qué emoción principal debe transmitir.
4. Qué sensación debe dejar en el público.
5. Qué universo visual sostiene su promesa.
6. Qué elementos visuales pueden repetirse.
7. Qué símbolos son coherentes con la marca.
8. Qué símbolos deben evitarse.
9. Qué estilo fotográfico representa mejor la experiencia.
10. Qué tipo de composición refuerza autoridad y deseo.

---

## DIRECCIÓN DEL BOARD

Crear una lámina visual premium, limpia y estratégica.

El board debe incluir:

1. **Logo o espacio de identidad**
   - nombre de la marca
   - símbolo o referencia de sistema visual si existe
   - aplicación limpia y profesional

2. **Paleta cromática**
   - colores principales
   - colores secundarios
   - acentos
   - neutros
   - uso emocional de cada color

3. **Atmósfera visual**
   - sensación general de la marca
   - energía
   - nivel de lujo
   - cercanía
   - autoridad
   - deseo

4. **Estilo fotográfico**
   - tipo de luz
   - encuadres
   - profundidad
   - sujetos
   - escenarios
   - momentos clave

5. **Texturas y materiales**
   - superficies
   - patrones
   - naturaleza
   - elementos físicos o digitales
   - detalles sensoriales

6. **Códigos gráficos**
   - líneas
   - formas
   - marcos
   - íconos
   - patrones
   - recortes
   - ritmo visual

7. **Tipografía y jerarquía**
   - titulares
   - subtítulos
   - textos comerciales
   - estilo editorial
   - contraste
   - respiración visual

8. **Aplicaciones de marca**
   - feed de Instagram
   - stories
   - reels covers
   - WhatsApp
   - web hero
   - presentación comercial
   - piezas promocionales

9. **Storytelling visual**
   - qué momentos debe mostrar la marca
   - qué emociones debe repetir
   - qué escenas activan deseo y confianza
   - qué visuales convierten atención en acción

## CRITERIOS VISUALES

El Visual Universe Board debe tener:

- composición premium
- jerarquía clara
- estética editorial
- coherencia cromática
- buen uso del espacio negativo
- lectura rápida
- sensación de sistema
- dirección aplicable
- códigos repetibles
- intención estratégica
- unidad visual

Debe sentirse como una guía visual de alto nivel, no como una imagen suelta.

---

## FORMATO DEL RESULTADO VISUAL

Generar una presentación tipo **brand universe board** o **visual identity board**.

Formato recomendado:

- fondo limpio
- composición en grilla editorial
- bloques visuales ordenados
- imágenes o escenas de referencia
- paleta cromática visible
- tipografías sugeridas
- texturas
- símbolos
- aplicaciones pequeñas
- detalles gráficos
- espacios de respiración

El board puede contener etiquetas breves como:

- Atmosphere
- Color System
- Photography
- Texture
- Graphic Codes
- Typography
- Social Media
- Web
- Brand Feeling

Pero no debe contener textos largos.

El resultado debe parecer una lámina profesional de dirección de arte creada por un estudio senior de branding.
