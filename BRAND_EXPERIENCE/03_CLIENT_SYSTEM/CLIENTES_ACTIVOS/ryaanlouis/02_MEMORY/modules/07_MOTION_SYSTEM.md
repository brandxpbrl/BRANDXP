# 07_MOTION_SYSTEM

## Función
Define cómo se mueve la marca.

## Principios
- Movimiento preciso.
- Ritmo cinematográfico.
- Transiciones limpias.
- Cámara con intención.
- Tipografía con timing editorial.
- Glow controlado.
- Profundidad visual.

## Evitar
- Efectos exagerados.
- Movimiento caótico.
- Zooms sin criterio.
- Transiciones infantiles.
