# MIRANDA EXPERIENCE — VISUAL SYSTEM

---

# VISUAL SYSTEM CORE

Miranda Experience no debe sentirse como una agencia turística.

Debe sentirse como:
- una experiencia emocional
- una presencia cálida
- un refugio premium
- una forma sofisticada de vivir Brasil
- un universo natural y elegante

Todo el sistema visual debe transmitir:
- calma
- lujo relajado
- autenticidad
- belleza natural
- conexión humana
- sofisticación emocional

---

# ATMÓSFERA GENERAL

El universo visual debe sentirse:

- Tropical Premium

- Calm Cinematic

Nunca debe sentirse:
- corporativo
- artificial
- saturado

- tecnológico frío

Debe sentirse:
- emocional
- suave
- natural
- cinematográfica

---

# REGLAS DE ILUMINACIÓN

## Priorizar:
- Golden Hour
- amanecer
- atardecer
- luz natural lateral
- reflejos cálidos
- sombras suaves
- brillo natural del sol

## Evitar:
- flashes fuertes
- luces frías
- contraste agresivo
- iluminación artificial evidente
- colores saturados

---

# COMPOSICIÓN VISUAL

Todo debe respirar.

El sistema visual usa:
- espacio negativo
- márgenes amplios
- composición editorial
- minimalismo elegante
- aire visual
- equilibrio emocional

---

# REGLAS DE COMPOSICIÓN

## SIEMPRE:
- usar composición limpia
- dejar respirar la imagen
- priorizar emoción sobre información
- crear profundidad visual
- usar grillas minimalistas
- mantener coherencia estética

## NUNCA:
- saturar
- usar demasiados elementos
- llenar todo el espacio
- usar textos enormes
- hacer diseños “promocionales baratos”
