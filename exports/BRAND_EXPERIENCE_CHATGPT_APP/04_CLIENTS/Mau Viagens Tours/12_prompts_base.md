# MANUAL FRAMEWORK ANALYSIS — Mau Viagens Tours

## 12. Prompts base

### Prompt base para logo

```text
Actúa como director creativo senior especializado en branding turístico premium-práctico. Diseña una propuesta de logo para Mau Viagens Tours, marca de turismo en Brasil que ayuda a viajeros a elegir, contratar y vivir experiencias de forma simple, confiable y humana.

Antes de diseñar, interpreta la entidad de marca: Brasil vibrante, confianza humana, claridad comercial, deseo de viajar y experiencia sin estrés.

Crear un sistema de logo con:
- logo principal
- logo horizontal
- versión compacta
- símbolo / isotipo
- monograma
- versión monocromática
- versión fondo claro
- versión fondo oscuro
- perfil Instagram
- versión WhatsApp

Dirección del símbolo:
- combinar M + ruta + ola + sol de forma sutil
- evitar avión genérico, maleta genérica o palmera obvia
- estilo tropical, confiable, simple, legible y profesional

Paleta:
#084928, #25654D, #619B1A, #0BA290, #F7D00B, #F9AA0B, #F3E89E, #FBFCFB

Entregar 3 rutas creativas con concepto, emoción, símbolo, color, tipografía, riesgos y aplicaciones.
```

### Prompt base para universo visual

```text
Crea un Visual Universe Board para Mau Viagens Tours.

Concepto visual madre: Brasil vibrante en manos confiables.

La lámina debe mostrar una identidad tropical, humana, clara y profesional. Incluir:
- paleta oficial
- fotografía recomendada
- composición
- texturas
- símbolos
- estilo de reels
- estilo de stories
- estilo de carruseles
- sellos de confianza
- dirección de logo
- dirección de WhatsApp visual

La estética debe evitar plantillas turísticas genéricas y debe transmitir deseo, claridad y confianza.
```

### Prompt base para storytelling board

```text
Crea un Storytelling Visual Board para Mau Viagens Tours basado en la idea madre: Brasil se vive mejor cuando viajás con confianza.

Mostrar la transformación:
- de duda a confianza
- de deseo a reserva
- de estrés a experiencia
- de querer viajar a saber con quién hacerlo

Incluir escenas de Brasil, agua, ruta, sol, amanecer, voz humana, WhatsApp, atención de Mauri y experiencia real.

La estética debe ser tropical, emocional, profesional y clara.
```
